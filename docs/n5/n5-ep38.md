# 38. 助詞：接触点「に」

表示动作的接触点。

名词A に B

- 車が （木|壁）に ぶつかりました。
- 车撞到了树、墙。
- 私は駅で 友達に 会いました。
- 我在车站遇到了朋友。 我偶然在车站遇到朋友
- 私は駅で 友達と 会いました。
- 我和朋友在车站相遇。 相互的动作，我和朋友都是在车站等待对方。

## 例句

- 昨日、デパートで友達に会いました。
- 昨天在百货公司遇到了朋友。
- 車が 壁に ぶつかりました。
- 车撞到了墙。
- 作品に 触らないでください。
- 请不要触摸作品。
- ボールが 頭に 当たりました。
- 球打到了头上。
