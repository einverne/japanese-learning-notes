# 46. ~ようになります / ~なくなります

变成~ / 变成不

- 動詞 辞書形 + ようになります 
- 動詞 ~~ない形~~ + なくなります

- い 形容詞 く + なります
- な 形容詞 に + なります
- 名詞 に + なります

- い 形容詞 く~~ない~~ + なくなります
- な 形容詞 じゃ~~ない~~ + なくなります
- 名詞 じゃ~~ない~~ なくなります

習慣的变化

- 社会人になってから、朝早く起きるようになりました。
- 成为社会人后，早上早起了。

能力的變化

- やっと日本語が話せるようになりました。
- 终于会说日语了。
