# 43. ～は…に あります（います）

表示某事物/某人或动物在(某处)。

人/动物/事物 名词 は + 场所（名词）に + あります/います

- 先生は 教室に います。
- 老师在教室里。
- 教科書は 机の上に あります。
- 教科书在桌子上。


- 机の上に りんごが あります。
- 桌子上有苹果。 （重点是苹果的存在，苹果对方还不知道桌子上有苹果）
- りんごは 机の上に あります。
- 苹果在桌子上。（重点是苹果的位置，苹果对方已经知道有苹果）

## 例句

- 郵便局は どこに ありますか。
- 邮局在哪里？
- 101 ビルは 台北に あります。
- 101 大楼在台北。
- 先生はどこにいますか。
- 老师在哪里？
- ペットの犬は 庭にいます。
- 宠物的狗在院子里。
