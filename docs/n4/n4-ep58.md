# 58. ~うとします

正要，想要~

動詞，意向形 と します

- 飲みます → 飲もう
- 食べます → 食べよう
- 来ます → 来(こ)よう
- します → しよう

動作的階段 A うとします B

- 切符を買おうとした時、財布を忘れた ことに気がつきました。
- 正要買票的时候，才发现忘了钱包。

拒否 A うとしません B

- 息子（むすこ）は 野菜を食べようとしません。
- 儿子不想吃蔬菜。

自己不能使用

- 私は 野菜 を 食べたくないです。
- 我不想吃蔬菜。

## 例文

- 出かけようとした時、友達から 電話がかかってきました。
- 正要出门的时候，朋友打电话来了。
- たくさんの人が マスクを買おうとして、店の前に 並んでいます。
- 很多人正要买口罩，排在店前。
