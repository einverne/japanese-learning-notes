# 53. 方法手段の「動詞て形」

动词 て 形 __B__

- 私は バスで 学校へ 行きます。
- 我坐公交车去学校。
- 私は 歩いて 学校へ 行きます。
- 我走路去学校。

走，表示方法

否定形的表现

- 何も 食べないで ダイエット します。
- 什么都不吃减肥。

## 例文

- 毎日 歩いて 学校へ 行きます。
- 每天走路去学校。
- スマホを 使って 体重を 管理しています。
- 用手机管理体重。
- 今回のテストでは辞書をひいて作文を書いてください。
- 这次考试请查字典写作文。
- お金を借りてマンションを買いました。
- 借钱买了公寓。
