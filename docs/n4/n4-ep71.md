# 71. 禁止形

禁止，不要

## 第一类动词

い 段 变成 う 段 + な

![mrGiCZ0nxO](https://pic.einverne.info/images/mrGiCZ0nxO.png)

## 第二类动词

- 教えます → 教えるな
- 食べます → 食べるな
- 起きます → 起きるな
- 見ます → 見るな
- 寝ます → 寝るな

## 第三类动词

- 来ます → 来(く)るな
- します → するな

- 廊下(ろうか)を走らないでください。
- 请不要在走廊上跑。
- 廊下を走るな。
- 不要在走廊上跑。

- 医者にタバコを吸うなと言われました。
- 医生告诉我不要抽烟。

## 例文

- 線路の近くで遊ぶな。
- 不要在铁路附近玩。
- 立入禁止は「中に入るな」という意味です。
- 禁止入内是指不要进入。
