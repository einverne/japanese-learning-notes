# 55. ~ように言います

指示，要求

动词 辞书形/ない形 + ように （言います/ 頼みます / 伝えます / お願いします / 指示します）

~と言います：某人说

- 今晩 すき焼を 作るよ。
- 今晚我要做日式火锅。
- 太郎、豆腐を買ってきて。

- お母さんは 太郎に すき焼を 作る と 言いました。
- 妈妈对太郎说，今晚要做日式火锅。

- お母さんは 太郎に 豆腐を買ってきて と 言いました。
- 妈妈对太郎说，去买豆腐。
- お母さんは 太郎に 豆腐を買ってくるように 言いました。
- 妈妈对太郎说，去买豆腐。

## 例文

- 母は弟に牛乳を買って来るように頼みました。
- 妈妈让弟弟去买牛奶。
