# 66. ～でも 全肯定

疑问词 A + 助词 + でも + B （肯定句）

- 私は嫌いな食べ物はありません。何でも食べられます。
- 我沒有討厭的食物。什麼都能吃。
- 明日なら、いつでも空いています。
- 明天的話，什麼時候都有空。
- 彼はやさしい性格で、誰にでも仲良くできます。
- 他性格溫和，和誰都能相處融洽。

- Q：東京へ旅行に行ったら、どこへ行きますか。
- ~~どこへでも行きたいです。~~
- いろいろな所へ行きたいです。
- A：想去很多地方。

## 例文

- 困ったときは、いつでも私に連絡してください。
- 有困難時，隨時聯絡我。
- 野口さんはどこででも寝ることができます。
- 野口先生無論在哪裡都能睡覺。
- 私は何でも食べられます。嫌いな物はありません。
- 我什麼都能吃。沒有討厭的東西。
- ドラえもんは誰でも知っている有名なアニメです。
- 哆啦A夢是誰都知道的有名動畫。
