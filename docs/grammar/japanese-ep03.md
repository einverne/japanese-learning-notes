# 大家的日本语 03 课文法解说

## 03 课 指示词 地方

- ここは  食堂(しょくどう) です。
- 这里是 食堂。

词：

- ここ 这里，靠近说话者
- そこ  那里，靠近对方
- あそこ ，那里，远方

哪里 どこ

- エレベーター は あそこです。
- 电梯在那里。

A6

- 问：これ は どこ の 車 ですか。
- 这是 哪里的车子？
- 答：これ は 日本 の 車（くるま）です。
- 这是日本的车子。
- の，表示所产

A7

- この ネクタイ は 1500（） 円です。
- 这条 领带 是 1500 日元。
