# 66. 助詞：全否定「も」

疑问词A 助词 も B(否定)   表示否定

- 今朝は 何も 食べませんでした。
- 今天早上什么都没吃。
- 明日は どこへも 行きません。 うちにいます。
- 明天不去任何地方。在家里。
- 今は 誰とも話したくないです。
- 现在不想和任何人说话。
- 妖怪は どこに もいません。人間の想像です。
- 妖怪不在任何地方。是人类的想象。

## 例文

- 教室に 誰も いません。
- 教室里没有任何人。
- 私は 何も 知りません。
- 我什么都不知道。
- 昨日は どこへも行きませんでした。
- 昨天什么地方都没去。
- 今は 誰にも会いたくないです。
- 现在不想见任何人。
