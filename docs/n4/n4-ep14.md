# 14. ～なさい

動詞~~ます~~ + なさい 

表示命令，请求，劝告，建议。

上 -> 下

- 食べなさい
- 野菜を食べなさい
- 吃蔬菜吧
- 早く寝なさい
- 早一点睡觉吧

命令，但是语气不会凶。不同于命令形。

其实是 します 的尊敬语。

します -> なさいます -> なさいませ

- おやすみなさい
- お帰りなさい

寒暄用语（あいさつ）

## 例文

- 野菜もちゃんと食べなさい。
- 蔬菜也要好好吃。
- テレビはもっと離れて見なさい。
- 离电视机远一点看。
- 面接のときは言葉遣いに気をつけなさい。
- 面试的时候，注意说话方式。
- 来週の金曜日までにレポートを出しなさい。
- 下周五之前交报告。
