# 65. ~まえに

~之前

- 動詞 辞書形 + まえに + B
  - A 之前，B
- 名詞 の + まえに


- 寝る前に 日記を書きます。
- 睡觉之前，写日记。   -> 日記を書いてから、寝ます。
- 食事の前に {お祈り|おいのり}をします。
- 吃饭之前，祈祷。   -> お祈りをしてから、食事をします。
- 三年前に 日本へ来ました。
- 三年前，来到了日本。

## 例文

- 友達のうちへ行く前に、電話をかけます。
- 去朋友家之前，打电话。
- プールに入る前に、準備運動をします。
- 进入游泳池之前、准备运动。
- 食事のまえに、手を洗います。
- 吃饭之前，洗手。
- 二年前に日本へ来ました。
- 两年前来到了日本。
