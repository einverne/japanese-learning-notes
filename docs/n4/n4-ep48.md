# 48. ~ようにしてください 

- ～ようにしてください：请做~，请努力做~

- 動詞 辞書形 + ようにしてください
- 動詞 ない形 + ようにしてください

- 毎日運動してください。
- 请每天运动。
- 毎日運動するようにしてください。
- 请努力每天运动。

- お酒を飲まないでください。
- 请不要喝酒。
- お酒を飲まないようにしてください。
- 请努力不要喝酒。

- 请借我把伞。
