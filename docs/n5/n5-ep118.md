# 118. 助詞：方面「に」

名詞 A に B

- ビタミン A は目の健康にいいです。
- 维他命 A 对眼睛的健康有好处。
- 花子ちゃんは お父さんに 似ています。
- 花子长得像爸爸。
- この辞書は漢字の読み方を調べるのに便利です。
- 这本词典查汉字的读音很方便。

## 例文

- タバコは体によくないです。
- 吸烟对身体不好。
- 桑原さんは お母さんによくにています。
- 桑原经常像妈妈一样。
- 吸血鬼は太陽の光に弱いです。
- 吸血鬼对太阳光很敏感。
- この動画は日本語の勉強にとてもいいです。
- 这个视频对学习日语很有帮助。
