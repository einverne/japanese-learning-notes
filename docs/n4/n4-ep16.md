# 16. ～てはいけません・～てはだめです

～てはいけません・～てはだめです  不可以做~

- 動詞 て形 + はいけません・はだめです

- タバコを吸ってはいけません。
- 不可以抽烟。
- お酒を飲んではいけません。
- 不可以喝酒。

- い形容词 くてはいけません
- な形容词 **で**はだめです
- 名词 **で**はいけません

- 品質が悪くてはいけません。
- 品质不好不行。
- 歌が下手ではだめです。
- 唱歌唱得不好不行。
- コピーではだめです。
- 不可以复印。

## 例文

- 嘘をついてはいけません。
- 不可以说谎。
- 他人の日記を読んではいけません。
- 不可以看别人的日记。
- 路上でタバコを吸ってはいけません。
- 不可以在路上抽烟。
- 車を運転するならお酒を飲んではいけません。
- 开车的话，不可以喝酒。
