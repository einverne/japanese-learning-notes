# 61. 名詞代替「の」

形式名词

普通形（な形容词，名词，动词的普通形） + の 


- 今 使っている *の(スマホ)* は、どこのスマホですか。
- 你现在用的是哪个手机？
- 当一个句子中要出现两个名词的时候

- 私が生まれた **の(町)**は、北海道の当別という*町*です。
- 我出生的城市是北海道的当别町。

- コーヒー、いかがですか。
- 咖啡怎么样？
- アリガトウございます。いただきます。
- 谢谢。我要了。
- 熱いの と 冷たいの と どちらが いいですか。
- 热的和冷的，哪个好？

试装中

- もっと大きい のは ありますか。
- 有更大的吗？

## 例文

- あそこで新聞を 読んでいるのは誰ですか。
- 那边在看报纸的是谁？
- 熱いのと冷たいのとどちらがいいですか。
- 热的和冷的，哪个好？
