
# 002. ～うえに

普通形（な ）+ うえに B 

不但 A 还 B

うえ（上）

豪華な うえに

この店の刺身は安いうえに おいしい。 这家店的刺身不但便宜还好吃。

## 例文

- 田中さんにはご馳走してもらったうえに、お土産までもらいました。
- 田中先生不仅请我吃饭，还送了我土特产。
- 彼の話は長いうえに、つまらない。
- 他的话不但冗长，还很无聊。
- 今なら入場料が半額のうえに、小学生以下の子供は無料で入ることができます。
- 现在的话，不但入场费半价，小学生以下的孩子还可以免费入场。
- この店の魚は新鮮な上に、値段も安い。
- 这家店的鱼不但很新鲜，价格也很便宜。
