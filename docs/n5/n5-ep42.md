# 42. ～に…が あります（います）

（某处）有（某事物/某人或动物）

场所（名词）に + 人/动物/事物 + が あります（います）

- に 表示位置
- が 表示焦点

- あそこに 犬 がいます。
- 那边有狗。
- あそこに 自動販売機 があります。
- 那边有自动贩卖机。

- 人/动物 需要使用 います。
- 物体/植物 需要使用 あります。

## 例句

- 冷蔵庫に ビールがあります。
- 冰箱里有啤酒。
- 公園に ブランコや滑り台があります。
- 公园里有秋千和滑梯。
- あそこに犬がいます。
- 那边有狗。
- 教室に誰もいません。
- 教室里没人。
