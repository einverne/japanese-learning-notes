# 60. ~さ

表示程度或狀態

- い 形容詞 去掉 い + さ
- な 形容詞 去掉な + さ

- 大きい -> 大きさ
- 長い -> 長さ
- 暑い -> 暑さ
- 楽しい -> 楽しさ

- 便利な -> 便利さ
- 静かな -> 静かさ
- 大変な -> 大変さ

- 前髪の長さはどうしますか。
- 前髮的長度怎麼樣？
- 子供が生まれて、子育ての大変さがよくわかりました。
- 孩子出生後，才明白育兒的辛苦。

## 例文

- 東京タワーの高さは333メートルです。
- 東京塔的高度是333公尺。
- この部屋の広さは10畳です。
- 這個房間的寬敞度是10畳。
