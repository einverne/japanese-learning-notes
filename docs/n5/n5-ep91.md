# 91. ください

名詞 A を 数量詞 ください

- 水を 一杯 ください。
- 请给我一杯水。
- もっと自由を ください。
- 请给我更多自由。
- じゃあ、このシャツを ください。
- 那么，请给我这件衬衫。
- アイスコーヒーを ください。
- 请给我一杯冰咖啡。


くれます -> くださいます

- 命令形：ください
- 拜托 くださいませんか

- 今 考えています。もう少し時間を くださいませんか。

## 例文

- すみません、きれいなお皿を一枚ください。
- 对不起，请给我一张干净的盘子。
- 瓶ビールを２本ください。コップは ４つお願いします。
- 请给我两瓶啤酒。杯子请给我四个。
- あのう、領収書をください。
- 请给我收据。
- 今 考えています。もう少し時間を くださいませんか。
- 现在在考虑。可以再给我一点时间吗？
