103. ～でしょう

~ でしょう 表示推测，相当于汉语的「吧」。

对不对？

普通形 + でしょう

1. 表示推断

- 明日は雨が降るでしょう。
- 明天会下雨吧。
- 3 月の東京はまだ寒いでしょう。
- 东京三月还是很冷吧。

2. 再次确认，同意要求

- 昨日、神社でお祭りがあったでしょう？
- 昨天在神社有祭典吧？
- この地図のアプリ、便利でしょう？
- 这个地图的应用很方便吧？

## 例文

- 今夜は雪が降るでしょう。
- 今晚会下雪吧。
- 彼の話はきっと本当でしょう。
- 他的话一定是真的吧。
- このラーメン、おいしいでしょう？
- 这个拉面很好吃吧？
- 先週、花火大会があったでしょう？
- 上周有烟花大会吧？
