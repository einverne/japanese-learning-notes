# 40. くださいます・~てくださいます

- くださいます
- 给我 ，别人給我
- ～てくださいます
- 请人（为自己）做某事

くれます

A(施惠者) は B(受惠者) に (名词 を / 动词 て) くださいます

- A 辈份高

- 部長は私に 英語を 教えてくださいました。
- 部长给我教了英语。

- ~~私は あなたに~~ 傘を 貸していただけますか。
- 请借我把伞好吗？
