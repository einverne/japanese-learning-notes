# 大家的日本语 28 课文法解说

## 文型 1

- 音楽を 聞きながら、食事します。
- 一边听音乐，一边吃饭。
- 聞く
- ながら，助词，表示同时进行（一边~，一边~）

<同时进行> A ます形 ながら、 B 。   B 主要动作，一边做A，一边做 B。

## 文型 3

- 地下鉄は 速いし、安いし、地下鉄で行きましょう。
- 地铁 又快、又便宜，搭地铁去吧。
- し，助词，表示列举理由。

<列举理由> 普通形（だ，だ）し、普通形（だ、だ）し、 C。

又 A，又 B，C。

- まだ時間もあるし、おなかも空いたし、映画の前にご飯を食べましょう。
